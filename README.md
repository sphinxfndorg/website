## Local Development

```bash
# Clone repository
git clone https://github.com/sphinxorg/website.git

# Enter project
cd website

# Start local server
python3 -m http.server 8000

# Alternative
npx serve

# Open browser
http://localhost:8000
```

## Deployment

### Cloudflare Pages

Automatic deployment is connected to the `main` branch.

```bash
git add .
git commit -m "update"
git push origin main
```

### Netlify

```bash
npm install -g netlify-cli
netlify deploy --prod
```

### Vercel

```bash
npm install -g vercel
vercel --prod
```
